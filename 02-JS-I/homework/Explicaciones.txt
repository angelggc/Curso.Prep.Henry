Variables: Las variables son cajas que guardan informacion y datos, puedo hacer una caja llamada "edad" y dentro guardar un numero. 
EJ: let edad = 21

String: Un string es un tipo de dato que tienen palabras o un texto, para que el lenguaje javascrip lo reconosca tiene que ser escrito entre comillas, tambien puede ser guardadas en variables. 
EJ: "Esto es un string"
let string = "Este es un string en una variable"

EJ: function funcionEj(argumento){
    return "Esta funcion devuelve un string junto al argumento => " + argumento
}
function funcion(){
    return "Esta funcion no tiene argumento :("

Booleanos: Un Boolean es un tipo de dato que puede tener 2 estados, true y false. Es facil de entender, true es Verdadero y false es Falso. Estos se pueden utilizar en condicionales y el resultado booleano se puede obtener de comparadores.
EJ: 1 == 1 //true
2 == 1 //false
